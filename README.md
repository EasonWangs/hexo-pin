# hexo-pin

**hexo-pin** is a masonry layout plugin used on hexo post page.

## Install

```bash
$ npm install hexo-pin --save
```

## Usage

```
{% pin 静夜思  [唐]李白%}
  床前明月光，
  疑是地上霜。
  举头望明月，
  低头思故乡。
{% endpin %}
```
